# 기능 정리

## 결과 판독
1. 볼 판독
2. 스트라이크 판독
3. 낫싱 판독
4. 정답 판독

## 출력
### 결과 출력
1. 볼, 스트라이크 개수 출력
2. 볼 출력
3. 낫싱 출력
4. 게임 종료 출력
### 게임 진행 가이드 출력
1. 게임 시작 문구 : "숫자 야구 게임을 시작합니다."
2. 게임 숫자 입력 : "숫자를 입력해주세요 : "
3. 게임 재시작 선택 문구 출력 : "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요."
## 입력
1. 숫자 입력
    - 3자리 수 입력
    - 재시작 & 종료 입력
    - 잘못된 입력 시 IllegalArgumentException 투척 후 게임 종료
